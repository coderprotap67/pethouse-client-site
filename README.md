# 🐾 PetHouse - Pet Adoption Platform

PetHouse is a full-stack pet adoption application designed to connect loving individuals with pets looking for forever homes. The platform provides a seamless experience for browsing available pets, submitting adoption requests, and managing pet listings.

🌐 **Live Website Link:** [https://pet-client-site.vercel.app](https://pet-client-site.vercel.app)

---

## ✨ Features

- **User Authentication:** Secure registration and login using JWT tokens and HTTP-only cookies.
- **Dynamic Pet Showcase:** Browse through detailed listings of pets with specific filters for search queries and species.
- **Adoption Request System:** Users can easily request to adopt a pet, track their requests, and receive updates.
- **Owner Dashboard:** Owners can create pet listings, view incoming requests for their pets, and accept or reject adoptions.
- **Responsive Interface:** Dark-themed modern layout completely responsive on mobile, tablet, and desktop views.

---

## 🛠️ Tech Stack

**Frontend:**
- Next.js (App Router, Client Components)
- Tailwind CSS (Styling)
- React Hot Toast (Notifications)
- Axios (API requests)

**Backend:**
- Node.js & Express.js
- MongoDB (Database via Atlas Cluster)
- JSON Web Token (JWT Authentication)
- Cookie-parser
